# 📦 Telecom Procurement Analytics

Projeto de análise de dados desenvolvido em Python utilizando dados reais anonimizados de compras e fornecedores de uma operação de telecomunicações.

O objetivo do projeto é transformar registros de compras em indicadores estratégicos para auxiliar no controle de custos, análise de fornecedores e tomada de decisões.

## 🎯 Objetivo

Analisar o histórico de compras para identificar:

- Valor total investido em materiais
- Principais fornecedores utilizados
- Produtos com maior impacto financeiro
- Ticket médio das compras realizadas
- Classificação de itens utilizando Curva ABC

## 🛠️ Tecnologias utilizadas

- Python
- Pandas
- Matplotlib
- Git e GitHub
- CSV - base de dados utilizada

## 📊 Indicadores

Durante a análise foram extraídos:

- Total de compras realizadas
- Valor total comprado
- Ticket médio por compra
- Ranking de fornecedores
- Produtos de maior relevância financeira
- Curva ABC de materiais
- ## 📈 Resultados da análise

A base analisada contém informações de compras realizadas na operação, permitindo identificar:

- Principais fornecedores por volume de compras
- Materiais responsáveis pelo maior investimento
- Distribuição dos gastos da operação
- Priorização de produtos utilizando Curva ABC

Essas análises permitem maior controle financeiro, planejamento de estoque e apoio nas decisões de compras.
